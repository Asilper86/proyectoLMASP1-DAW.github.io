    • Ejercicios para prácticar C.

    Aquí se muestran 5 ejercicios para la práctica del lenguaje C:

    1. Intercambiar dos números

      2. Escribe un programa en C que pida dos números al usuario y luego los intercambie utilizando una variable temporal.

      Instrucciones:
      Solicita dos números enteros.
      Intercambia los valores de esos dos números sin perder el valor original de ninguno de ellos.
      Muestra los números después de haberlos intercambiado.


    2. Contar dígitos de un número

      3. Escribe un programa en C que cuente cuántos dígitos tiene un número ingresado por el usuario.

      Instrucciones:
      Solicita un número entero.
      Utiliza un bucle para contar cuántos dígitos tiene el número.
      El programa debe funcionar incluso con números negativos.


    3. Matriz transpuesta

      4. Escribe un programa en C que calcule la transpuesta de una matriz. La transpuesta de una matriz es aquella que se obtiene intercambiando sus filas y columnas.

      Instrucciones:
      Solicita al usuario ingresar el número de filas y columnas de la matriz.
      Luego pide los valores de la matriz.
      Calcula la transpuesta de la matriz e imprímela.


    4. Promedio de un arreglo

      5. Escribe un programa en C que calcule el promedio de los elementos de un arreglo de números enteros.

      Instrucciones:
      Pide al usuario la cantidad de elementos del arreglo.
      Solicita los valores de los elementos del arreglo.
      Calcula y muestra el promedio de los valores.


    5. Buscar un valor en un arreglo

      6. Escribe un programa en C que busque un valor en un arreglo y determine si ese valor está presente o no.

      Instrucciones:
      Pide al usuario ingresar un número de elementos en el arreglo.
      Solicita los elementos del arreglo.
      Luego, pide un valor a buscar.
      El programa debe indicar si el valor está presente o no en el arreglo.
