    • Ejercicios para prácticar CSS.

    Aquí se muestran las soluciones de los 5 ejercicios para la práctica del lenguaje CSS:

    1. Diseñar una página con un fondo de color y texto centrado

      2. Crea una página web con un fondo de color y coloca un título y un párrafo de texto centrado tanto horizontal como verticalmente.

      Instrucciones:
      Define un color de fondo para la página.
      Centra el texto dentro de la página, tanto vertical como horizontalmente.



    2. Crear una tarjeta con sombra y borde redondeado

      3. Crea una tarjeta que tenga un borde redondeado, una sombra y un título dentro de ella. La tarjeta debe tener un espacio de relleno (padding) y un margen adecuado.

      Instrucciones:
      Define una tarjeta con borde redondeado usando la propiedad border-radius.
      Agrega una sombra a la tarjeta usando la propiedad box-shadow.
      Agrega un poco de relleno y márgenes.



    3. Crear un menú de navegación horizontal

      4. Crea un menú de navegación que sea horizontal, con enlaces que cambien de color cuando se pase el mouse por encima de ellos (hover).

      Instrucciones:
      Utiliza una lista (<ul>) para crear los elementos del menú.
      Los enlaces deben cambiar de color cuando el usuario pase el ratón sobre ellos.



    4. Diseñar un formulario con campos de entrada y botones estilizados

      5. Crea un formulario con varios campos de entrada (input) y un botón. Asegúrate de que los campos y el botón tengan un estilo adecuado, con un fondo, bordes y márgenes definidos.

      Instrucciones:
      Aplica estilos a los campos de entrada (input) y al botón (button).
      Asegúrate de que el botón tenga un color de fondo diferente cuando el usuario pase el ratón por encima (hover).



    5. Crear una cuadrícula de imágenes

      6. Crea una cuadrícula de imágenes con 3 columnas. Asegúrate de que las imágenes tengan el mismo tamaño y se ajusten bien dentro de la cuadrícula, incluso si el tamaño de la pantalla cambia.

      Instrucciones:
      Utiliza la propiedad display: grid para crear la cuadrícula.
      Asegúrate de que las imágenes se ajusten bien a la cuadrícula utilizando object-fit y grid-template-columns .
