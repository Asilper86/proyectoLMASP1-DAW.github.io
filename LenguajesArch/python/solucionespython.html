    • Ejercicios para prácticar Python.

    Aquí se muestran las soluciones a los 5 ejercicios para la práctica del lenguaje Python:

    1. Contador de Vocales

      2. Escribe un programa que reciba una cadena de texto del usuario y cuente cuántas vocales (a, e, i, o, u) contiene.

      Instrucciones:
      Pide al usuario una cadena de texto.
      Utiliza un bucle for para recorrer la cadena y cuenta cuántas veces aparecen las vocales.
      Muestra el resultado al usuario.



    2. Palíndromo

      3. Escribe un programa que determine si una palabra o frase es un palíndromo. Un palíndromo es una palabra o frase que se lee igual hacia adelante y hacia atrás (ignorando espacios, comas y tildes).

      Instrucciones:
      Pide al usuario una palabra o frase.
      Elimina los espacios y caracteres no alfabéticos.
      Comprueba si la cadena es igual a su reverso.



    3. Adivinar el Número

      4. Escribe un juego en el que la computadora elija un número aleatorio entre 1 y 100 y el usuario deba adivinarlo. El programa debe indicar si el número introducido es mayor o menor al número objetivo y seguir pidiendo intentos hasta que el usuario lo adivine.

      Instrucciones:
      Usa el módulo random para generar un número aleatorio.
      Pide al usuario que adivine el número y da pistas si el intento es mayor o menor.
      El juego termina cuando el usuario adivina el número.



    4. Contar Palabras en una Frase

      5. Escribe un programa que reciba una frase del usuario y cuente cuántas palabras tiene. Asume que las palabras están separadas por espacios.

      Instrucciones:
      Pide al usuario una frase.
      Divide la frase en palabras usando el método .split().
      Cuenta cuántas palabras contiene la lista resultante.



    5. Fibonacci

      6. Escribe un programa que calcule los primeros N números de la secuencia de Fibonacci, donde cada número es la suma de los dos anteriores (0 y 1 al principio). El programa debe pedir al usuario cuántos números de la secuencia desea ver.

      Instrucciones:
      Pide al usuario cuántos números de Fibonacci quiere ver.
      Usa un bucle for para calcular y mostrar los primeros N números de la secuencia.
