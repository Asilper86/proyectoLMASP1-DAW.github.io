    • Ejercicios para prácticar C++.

    Aquí se muestran 5 ejercicios para la práctica del lenguaje C++:

    1. Escribe un programa en C++ que pida al usuario tres números y luego imprima el mayor de ellos.

      Instrucciones:
      Solicita tres números al usuario.
      Usa una estructura condicional para determinar cuál es el mayor de los tres.


    2. Suma de los Primeros N Números

      3. Escribe un programa en C++ que calcule la suma de los primeros N números enteros positivos.

      Instrucciones:
      Solicita al usuario un número N.
      Calcula la suma de los números desde 1 hasta N usando un bucle for.


    3. Palíndromo

      4. Escribe un programa en C++ que determine si una palabra ingresada por el usuario es un palíndromo. Un palíndromo es una palabra que se lee igual de izquierda a derecha que de derecha a izquierda.

      Instrucciones:
      Solicita una palabra al usuario.
      Verifica si la palabra es un palíndromo.


    4. Promedio de una Lista de Números

      5. Escribe un programa en C++ que calcule el promedio de una lista de números que el usuario ingresa.

      Instrucciones:
      Solicita al usuario cuántos números desea ingresar.
      Luego, solicita esos números uno por uno y calcula el promedio.


    5. Contar las Apariciones de un Carácter

      6. Escribe un programa en C++ que cuente cuántas veces aparece un carácter específico en una cadena de texto proporcionada por el usuario.

      Instrucciones:
      Solicita una cadena de texto y un carácter.
      Cuenta y muestra cuántas veces aparece el carácter en la cadena.
