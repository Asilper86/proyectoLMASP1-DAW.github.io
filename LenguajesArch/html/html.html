    • Ejercicios para prácticar HTML.

    Aquí se muestran 5 ejercicios para la práctica del lenguaje HTML:

    1. Crear una página básica con título y párrafos

      2. Escribe una página web en HTML que contenga un título y varios párrafos de texto.

      Instrucciones:
      Crea un archivo HTML con un título en la parte superior (en la etiqueta <title>).
      Agrega al menos 3 párrafos (<p>) con contenido de tu elección.</p>

    2. Crear una lista ordenada y desordenada

      3. Crea una página HTML que contenga dos listas:

      Una lista ordenada (numerada).
      Una lista desordenada (con viñetas).


    3. Formulario de contacto simple

      4. Crea un formulario de contacto en HTML donde los usuarios puedan ingresar su nombre, correo electrónico y un mensaje.

      Instrucciones:
      Usa las etiquetas <form>, <input>, y <textarea> para crear un formulario.
      Asegúrate de incluir un botón para enviar el formulario.


    4. Agregar imágenes a una página web

      5. Escribe una página HTML que contenga una imagen. Asegúrate de usar la etiqueta <img> y de establecer el atributo src correctamente.

      Instrucciones:
      Usa una imagen que se encuentre en tu computadora o usa una URL de una imagen en línea.
      La imagen debe tener un tamaño adecuado y un texto alternativo (atributo alt).


    5. Crear una tabla simple

      6. Crea una tabla HTML que tenga al menos 3 filas y 2 columnas. Asegúrate de usar las etiquetas adecuadas para las celdas de la tabla.

      Instrucciones:
      Usa las etiquetas <table>, <tr>, <td>, y <th> para crear la tabla.
      Incluye un encabezado en la primera fila utilizando <th> y agrega datos a las celdas.
