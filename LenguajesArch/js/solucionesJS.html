    • Ejercicios para prácticar JavaScript.

    Aquí se muestran las soluciones de los 5 ejercicios para la práctica del lenguaje JavaScript:

    1. Contar las palabras de una frase

      2. Escribe un programa en JavaScript que cuente cuántas palabras tiene una frase ingresada por el usuario.

      Instrucciones:
      Solicita una frase al usuario.
      Divide la frase en palabras y cuenta cuántas palabras contiene.



    2. Generar una secuencia de Fibonacci

      3. Escribe un programa en JavaScript que imprima los primeros N números de la secuencia de Fibonacci.

      Instrucciones:
      Solicita al usuario un número N.
      Muestra los primeros N números de la secuencia de Fibonacci.



    3. Verificar si un número es primo

      4. Escribe un programa en JavaScript que determine si un número ingresado por el usuario es primo.

      Instrucciones:
      Solicita un número entero al usuario.
      Verifica si el número es primo (solo divisible por 1 y por sí mismo).



    4. Invertir una cadena de texto

      5. Escribe un programa en JavaScript que invierta una cadena de texto ingresada por el usuario.

      Instrucciones:
      Solicita una cadena de texto al usuario.
      Muestra la cadena invertida.



    5. Generar una tabla de multiplicar

      6. Escribe un programa en JavaScript que genere la tabla de multiplicar de un número ingresado por el usuario.

      Instrucciones:
      Solicita un número al usuario.
      Muestra la tabla de multiplicar de ese número, desde el 1 hasta el 10
